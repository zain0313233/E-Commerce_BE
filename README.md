## Getting Started

1. Clone the repository:
    ```bash
    git clone https://github.com/zain0313233/E-Commerce_BE.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Set up environment variables (see `.env.example`).
4. Start the server:
    ```bash
    npm run dev
    ```
